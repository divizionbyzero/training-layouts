CSS SLIDER

  • transition

    A shorthand property for setting the four transition properties into a single property.

    Default value: all 0 ease 0

    transition: property duration timing-function delay | initial | inherit;

    transition-property

    The transition-property property specifies the name of the CSS property the transition effect is for (the transition effect will start when the specified CSS property changes).

    transition-property: none | all | property| initial | inherit;

  • transition-duration

    Specifies how many seconds (s) or milliseconds (ms) a transition effect takes to complete.

    transition-duration: time | initial | inherit;

    transition-delay

    Specifies when the transition effect will start.

    transition-delay: time | initial | inherit;

  • transition-timing-function

    Specifies the speed curve of the transition effect. This property allows a transition effect to change speed over its duration.

    transition-timing-function: ease | linear | ease-in | ease-out | ease-in-out | cubic-bezier() | initial | inherit;

    Property values on w3schools.com

  • animation

    shorthand property for six of the animation properties.

    Default value: none 0 ease 0 1 normal none running

    animation: name duration timing-function delay iteration-count direction fill-mode play-state;

    More about on w3schools.com

  • transformation

    The transform property applies a 2D or 3D transformation to an element. This property allows you to rotate, scale, move, skew, etc., elements.

    Default value: none

    transform: none | transform-functions | initial | inherit;
